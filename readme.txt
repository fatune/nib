GMT_XYEdit
==========

Программа для быстрого и простого редактирования xy файлов, которые в
последствии планируется отрисовывать программой GMT (Generic Mapping Tool).

Программа позволяет реактировать:
* sites
* lines/polygons
* labels
* выносные точки

Пока что упор в программе будет делаться на редактирование надписей (включая
изогнутые) и выносных точек.

Программа должна парсить уже существующие надписи.
Программа иметь возможность парсить существущие выносыне точки

Выносные точки
==============

Точки с примечаниями или что-то подобное.
Пример --- облако эпицентров в центре карты, и от каждого эпицентра идет линия к
мячику с фокальным механизмом расположенному с краю карты.

Выносные линии к уже существующим точкам
========================================

Есть файл sites.xy
120 50
122 55
126 64
118 55

При начале создания выносных точек генерируется файл sites_$имя$.xy c таким же
содержанием как и sites.xy
И файл sites_$имя$_lines.xy
120 50
120 50
>
122 55
122 55
>
126 64
126 64
>
118 55
118 55

Редактируя выносные точки (перемещая их по карте) файл sites.xy остается
нетронутым. 

При редактировании уже существующих выносных точек (т.е. файла sites.xy
sites_$имя$.xy и sites_$имя$_lines.xy), в котором файл sites_$имя$.xy имеет вид

Y X длинная-строка-для-каких-то-других-целей 

изменению в этом файле подвергаются только X Y. остальная часть строки остается 
без изменений.  Порядок строк в файле также остается без изменений.
